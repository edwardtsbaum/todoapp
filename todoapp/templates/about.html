About This App

This is a To-Do application that works with a Flask framework. This is a beginner application and is not by any means the final product. Criticize lightly please.

Specifications

  • You can sign up and create a user profile.
  • You can login.
  • When logged in you can only see the logout route.
  • Cannot access the todoapp without logging in.
  • you can enter tasks and due dates.
  • You can edit your tasks.
  • You can mark your tasks complete.
  • You can mark your task incomplete.
  • You can delete tasks.

Features that didn't make it

  • You cannot sort the todos.
  • There is a bug that lets you set todos before the current date.
  • You cannot edit the due dates.
  • Did not include profile specific todos.
  • the forgot password feature.

You can click on the Todo App to go to your home page.
